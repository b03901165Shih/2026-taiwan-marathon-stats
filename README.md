## Main features

| 功能 | 操作 | 輸出 |
|------|------|------|
| **分布圖** | 選賽事+賽別+分組 | 5分鐘區間完賽時間長條圖 |
| **查排名** | 輸入 `HH:MM:SS` | 全場+性別+細組精準 **PR%** |
